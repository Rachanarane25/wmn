About the Course

Wireless systems, mobility, simulation, and performance

This portfolio documents practical learning from the Wireless and Mobile Networks course, including wireless propagation, scanning, handover, 5G core setup, UE registration, network slicing, MEC latency analysis, and an applied smart healthcare use case.

Course Outcomes

Learning outcomes demonstrated in the portfolio

CO 1

Analyze Wireless Behavior

Compare propagation models, scanning approaches, and handover behavior using measurable wireless performance indicators.

CO 2

Configure 5G Core Components

Deploy Open5GS components, configure subscribers, integrate UERANSIM gNB and UE, and validate PDU session establishment.

CO 3

Evaluate Network Performance

Measure latency, throughput, packet delivery, registration time, and QoS behavior across simulated and 5G SA scenarios.

CO 4

Apply Network Slicing

Design service-specific eMBB, URLLC, and mMTC slices with differentiated QoS policies and isolated traffic behavior.

CO 5

Use Edge Computing Concepts

Demonstrate how MEC reduces response time for latency-sensitive mobile services such as real-time healthcare monitoring.

CO 6

Document Technical Work

Organize aims, theory, procedures, screenshots, results, conclusions, and references into a professional engineering portfolio.
